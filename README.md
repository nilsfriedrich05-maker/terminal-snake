# Terminal Snake

A classic Snake game that runs directly in your terminal, written in Python.

## Requirements

- Python 3
- A terminal that supports curses (macOS Terminal, iTerm2, etc.)

## How to run

```bash
python3 snake.py
```

## Controls

| Key | Action |
|-----|--------|
| Arrow keys or WASD | Move snake |
| Q | Quit |
| Enter | Start / Restart |

## Features

- Highscore saved between sessions
- Snake gets faster as your score increases
- Colorful gradient snake rendering
